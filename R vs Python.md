

### Index
R is based 1
```R
numbers = c(1, 2, 3, 4, 5)
numbers[0] # -> NULL
numbers[1] # -> 1
numbers[2] # -> 2
numbers[5] # -> 5
numbers[6] # -> NULL
```
Python is based 0
```Python
numbers = [1, 2, 3, 4, 5]
numbers[0] # -> 1
numbers[1] # -> 2
numbers[2] # -> 3
numbers[5] # -> IndexError: list index out of range
```
